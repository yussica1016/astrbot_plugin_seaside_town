## 系统

- **天气**：晴天·多云·雨天·大雾·暴风·满天星。同一天天气固定。影响可去的地方和场景描写。
- **时间**：清晨·午后·傍晚·夜晚。跟随真实时间。影响NPC问候和场景氛围。
- **拾贝**：普通/稀有/极稀有三级。有机会捡到漂流瓶。
- **钓鱼**：随机钓到鱼/宝贝/垃圾。
- **随机事件**：每到一个地方20%概率触发。
- **NPC记忆**：记录见面次数。
- **自动漫游**：AI伴侣可以自己逛小镇，记录在日记里。

## 安装

1. 将文件夹放入 `AstrBot/data/plugins/`
2. 重启或重载插件
3. 发 `小镇` 开始
